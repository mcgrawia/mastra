# Workflow.execute()

The `.execute()` method executes a workflow directly and returns the output, allowing you to run a workflow without creating a separate run instance.

## Usage example

```typescript showLineNumbers copy
await workflow.execute({
  inputData: {
    value: "hello world"
  }
});
```
